# fusioncore_husky

FusionCore sensor fusion demo for the Clearpath Husky A200.
Fuses wheel odometry, Microstrain 3DM-GX5-25 IMU, and u-blox F9P GPS
into a continuous global-frame odometry estimate at 100 Hz.

See [demos_docs/fusioncore_husky.md](../../demos_docs/fusioncore_husky.md) for
documentation and run instructions.
